# Copy, cut & paste

Every shift block supports **Copy**, **Cut**, and **Paste** via the right-click context menu.

[→ Try it live](/demo)

## Right-click menu

Right-click any block to access:

| Action | Behaviour |
|--------|-----------|
| **Edit shift** | Opens the edit modal |
| **Copy shift** | Copies block to clipboard — original stays on grid |
| **Cut shift** | Copies to clipboard AND immediately removes from grid |
| **Delete shift** | Opens delete confirmation |

## Paste a copied shift

After copying or cutting, click the `+` button on any empty cell to paste the shift at that position. The pasted shift gets a new `id` but retains all other properties.

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `Delete` / `Backspace` | Delete focused block |
| `Arrow keys` | Move focused block by snap interval |

## Programmatic copy

You can control the copied shift externally:

```tsx
const [copiedShift, setCopiedShift] = useState<Block | null>(null)

<Scheduler
  copiedShift={copiedShift}
  setCopiedShift={setCopiedShift}
/>

// Programmatically copy a shift:
setCopiedShift(someBlock)
```
