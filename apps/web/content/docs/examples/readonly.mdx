# Read-only mode

Read-only mode disables all interactions — no dragging, resizing, clicking, or editing. Useful for display dashboards or shared schedule views.

[→ Try it live](/demo)

```tsx
<Scheduler
  readOnly
  categories={categories}
  employees={employees}
  shifts={shifts}
  onShiftsChange={() => {}}   // no-op, won't be called
/>
```

## What's disabled in read-only

- Drag and drop
- Block resize handles
- Right-click context menu
- Double-click to edit
- Add-shift `+` buttons
- All keyboard editing shortcuts

## What still works

- View switching (day/week/month/year/list/timeline)
- Navigation (prev/next period)
- Hover popover (shift details)
- Zoom controls
- Scroll to now
