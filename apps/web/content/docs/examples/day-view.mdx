# Day view

The day view shows a single day as a horizontal timeline. Users can swipe left/right to navigate between days.

[→ Try it live](/demo)

```tsx
<Scheduler
  categories={categories}
  employees={employees}
  shifts={shifts}
  onShiftsChange={setShifts}
  initialView="day"
  config={createSchedulerConfig({ initialScrollToNow: true })}
/>
```

## Navigation

- **Swipe left/right** on touch devices to go to the next/previous day
- **Arrow keys** (`←` `→`) when no block is focused
- **Date picker** in the toolbar header

## Cross-day drag

In day view, dragging a block past the **right edge** moves it to the next day. Dragging past the **left edge** moves it to the previous day.
