# Week view

The week view shows 7 days as columns. This is the default view.

[→ Try it live](/demo)

```tsx
<Scheduler
  initialView="week"
  // ... other props
/>
```

## Scrolling

The week view uses a scroll buffer of ±15 days. As you scroll to the edges, the buffer automatically loads the next/previous period without any loading spinner.

## Zoom

Use the zoom control (or `zoom` prop) to see more or fewer hours per column:

```tsx
const [zoom, setZoom] = useState(1)

<Scheduler
  zoom={zoom}
  setZoom={setZoom}
  // ...
/>
```

Zoom levels: `0.5` (zoomed out) → `2.0` (zoomed in). Higher zoom shows more detail and wider time slots.
