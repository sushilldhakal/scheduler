# Zoom levels

Zoom controls how wide each hour column is — zooming in shows more detail and wider slots, zooming out shows more time at once.

[→ Try it live](/demo)

## Uncontrolled (toolbar controls)

By default, the toolbar provides zoom in/out buttons. No additional props needed:

```tsx
<Scheduler ... />
// Toolbar zoom buttons work out of the box
```

## Controlled zoom

```tsx
const [zoom, setZoom] = useState(1)

<Scheduler
  zoom={zoom}
  setZoom={setZoom}
  ...
/>

// Zoom buttons
<button onClick={() => setZoom(z => Math.min(z + 0.25, 2))}>Zoom in</button>
<button onClick={() => setZoom(z => Math.max(z - 0.25, 0.5))}>Zoom out</button>
```

## Zoom levels

| Value | Effect |
|-------|--------|
| `0.5` | Very compact — 2-hour minimum slot |
| `0.75` | Compact |
| `1.0` | Default — 1-hour slots |
| `1.25` | Detailed — 45-minute slots visible |
| `1.5` | More detailed |
| `2.0` | Maximum — 30-minute slots clearly visible |
