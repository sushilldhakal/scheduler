# Scroll to now

The scheduler can automatically scroll to the current time on load, and show a live pulsing indicator on the current-time line.

[→ Try it live](/demo)

## Enable on load

```tsx
import { createSchedulerConfig } from '@sushill/shadcn-scheduler'

const config = createSchedulerConfig({
  initialScrollToNow: true,
})

<Scheduler config={config} ... />
```

## Programmatic scroll

Trigger scroll-to-now from a button using the `scrollToNowRef`:

```tsx
import { useRef } from 'react'

const scrollToNowRef = useRef<(() => void) | null>(null)

<button onClick={() => scrollToNowRef.current?.()}>
  Jump to now
</button>

<Scheduler
  scrollToNowRef={scrollToNowRef}
  config={config}
  ...
/>
```

## Live time indicator

When `initialScrollToNow` is true, the current time is marked with a vertical red line and a pulsing dot. The indicator updates every minute automatically.
