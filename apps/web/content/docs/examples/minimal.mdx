# Minimal setup

The absolute minimum to get a scheduler on screen. One category, one employee, no shifts.

[→ Try it live](/demo)

```tsx
'use client'
import { useState } from 'react'
import { Scheduler } from '@sushill/shadcn-scheduler'

const categories = [
  { id: 'c1', name: 'Team', kind: 'category' as const, colorIdx: 0 },
]
const employees = [
  { id: 'e1', name: 'Alex', kind: 'employee' as const, categoryId: 'c1', colorIdx: 0 },
]

export default function MinimalScheduler() {
  const [shifts, setShifts] = useState([])
  return (
    <div className="h-[500px]">
      <Scheduler
        categories={categories}
        employees={employees}
        shifts={shifts}
        onShiftsChange={setShifts}
        initialView="week"
      />
    </div>
  )
}
```

This gives you: drag & drop, right-click context menu, add-shift by clicking empty cells, and full keyboard support — all out of the box.
