# Multi-day view

The multi-day layout renders multiple days side by side as columns, similar to a compressed week view. Users scroll horizontally to navigate across time.

[→ Try it live](/demo)

```tsx
<Scheduler
  initialView="week"    // week view with bufferDays controls how many days appear
  bufferDays={7}        // ±7 days of buffer on each side
  // ... other props
/>
```

## Wide screen support

On screens wider than 2000px the scheduler automatically adjusts the date buffer so the edge-load zones never overlap, preventing the flicker/loop that would otherwise occur when the entire buffer fits in the viewport.
