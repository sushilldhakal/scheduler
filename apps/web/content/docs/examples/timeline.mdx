# Timeline view

The timeline view renders resources as rows and time as columns, with blocks spanning horizontal time ranges. Best suited for TV guides, venue bookings, and project Gantt charts.

[→ Try it live](/demo)

```tsx
<Scheduler
  initialView="timeline"
  config={createTvConfig()}
  // ... other props
/>
```
