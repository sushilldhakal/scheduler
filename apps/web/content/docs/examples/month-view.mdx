# Month view

The month view shows a full calendar month. Each day cell displays shift indicators — click a day to see its details.

[→ Try it live](/demo)

```tsx
<Scheduler
  initialView="month"
  // ... other props
/>
```

## Shift indicators

Each day shows coloured dots for shifts. The colour matches the category's `colorIdx`.

## Clicking a day

Clicking a day in month view navigates to the day view for that date.
